Well for starters I would like to develop different "personality types". Rather than the A.I. all following the same formula (hiring 1/3, training 1/3, building 1/3, X likelihood of attacking, etc.) I'd like to create various personalities to make things more interesting for the player.

Here is the example of how I would like the A.I. players for my presented simulation to function (note : they don't currently function this way, this is one of the first features I would like to add, along with the human player of course.

Name : Name of leader, pretty self explanatory

T%, H%, B% : Each turn the AI players must take an action. If they do not go to war (or transfer troops) they have the choice to either Train, Hire or Build (Economy), these percentages show the likelihood of each for the given player.

ASN = Attack Strngth Needed. A player will only even consider attack if he has this percentage of his potential victim's amount of troops.

AA = Agressiveness Amount. The likelihood of attack with go up or down a percentage based on this number.
